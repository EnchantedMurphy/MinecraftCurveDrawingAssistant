# Minecraft 绘制辅助工具

一个纯前端 HTML 工具，帮助在 Minecraft 中沿贝塞尔曲线铺设方块（铁路、墙壁、道路等）。

## 使用方法

- 方法一（在线使用）：[Minecraft绘制辅助工具](https://enchantedmurphy.github.io/MinecraftCurveDrawingAssistant/)
- 方法二（本地部署）：直接用浏览器打开 `index.html`，无需安装任何依赖。

## 界面布局

- **最左侧图标栏**：收展按钮、曲线绘制标签、GitHub 链接、明暗模式切换
- **左侧可收起面板**：标题"绘制辅助工具"、坐标输入、曲线调节、方块路径模式、坐标输出
- **右侧画布**：铺满剩余空间，左上角显示项目标题及版本号，右上角缩放控件，右下角图例

支持深色/浅色模式一键切换，侧边栏可收起以最大化画布空间。

## 功能说明

### 输入参数

| 参数 | 说明 |
|------|------|
| 起点/终点 X、Z | Minecraft 世界坐标（Y 轴忽略，仅处理水平面） |
| 切线角度 | 0° = +X 方向，90° = +Z 方向，支持 0°–360° 全向，滑块与数字框双向联动 |
| 起点/终点把手长度 | 控制贝塞尔曲线控制点的偏移距离，值越大该端越贴近切线方向 |

### 曲线调节

分别输入起点和终点的把手长度，画布实时预览曲线变化。虚线显示贝塞尔控制把手位置，黄色点标记控制点位置。

### 方块路径模式

- **无对角连接**（4 连通）：路径仅使用上下左右相邻方块，适合铁路轨道
- **有对角连接**（8 连通）：优先走对角步，使用方块数更少，适合墙壁等结构

### 可视化画布

- 蓝色方块：离散化后的方块路径
- 橙色曲线：原始贝塞尔曲线
- 紫色点：起点（S）和终点（E）
- 黄色点：贝塞尔控制点及控制把手虚线
- 鼠标拖拽平移视图，滚轮或滑块缩放，点击"重置视图"居中显示

### 坐标输出

点击"生成曲线"后，所有方块坐标按顺序显示在文本框中（每行 `X Z`），文本框下方显示方块总数，可直接复制用于 Minecraft 建造。

## 算法说明

1. 根据端点坐标、切线角度和把手长度，构造三次贝塞尔曲线（四个控制点）
2. 参数化追踪：维护当前方块坐标，沿曲线密集采样推进，每步选择 8 邻居中欧氏距离最近目标点的方向，自然在对角方向产生对角步
3. 4 连通模式：对 8 连通路径中的对角步插入一个轴对齐中间方块，插入轴根据下一步方向决定
