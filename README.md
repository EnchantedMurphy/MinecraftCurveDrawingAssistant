# Minecraft 曲线绘制辅助

一个纯前端 HTML 工具，帮助在 Minecraft 中沿贝塞尔曲线铺设方块（铁路、墙壁、道路等）。

## 使用方法

直接用浏览器打开 `index.html`，无需安装任何依赖。

## 功能说明

### 输入参数

| 参数 | 说明 |
|------|------|
| 起点/终点 X、Z | Minecraft 世界坐标（Y 轴忽略，仅处理水平面） |
| 切线角度 | 0° = +X 方向，90° = +Z 方向，控制曲线在端点处的走向 |
| 起点/终点把手长度 | 控制贝塞尔曲线控制点的距离，值越大曲线在该端越"直" |

### 曲线调节

拖动两个滑块分别调整起点和终点的"把手长度"，即三次贝塞尔曲线的控制点偏移量。实时预览曲线变化，黄色虚线显示控制把手。

### 方块路径模式

- **无对角连接**（4 连通）：路径仅使用上下左右相邻方块，适合铁路轨道
- **有对角连接**（8 连通）：优先走对角步，使用方块数更少，适合墙壁等结构

### 可视化画布

- 橙色曲线：原始贝塞尔曲线
- 绿色方块：离散化后的方块路径
- 紫色点：起点（S）和终点（E）
- 黄色点：贝塞尔控制点
- 鼠标拖拽平移视图，滚轮缩放，或使用缩放滑块

### 坐标输出

生成后，所有方块坐标按顺序显示在右侧文本框，格式为每行 `X Z`，可直接复制用于 Minecraft 建造参考。

## 算法说明

1. 根据端点坐标、切线角度和把手长度，构造三次贝塞尔曲线的四个控制点
2. 对曲线密集采样（1000 点），将每段用 Bresenham 直线算法光栅化为整数方块坐标
3. 4 连通模式下，将 8 连通路径中的对角步骤拆分为两步轴对齐移动
